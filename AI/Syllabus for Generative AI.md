### **Module 1: Machine Learning Project**
---
#### **Project 1: Customer Churn Prediction**

**Objective:**
Develop a model to predict customer churn in a telecommunications company.

**Dataset:**
[Kaggle Telco Customer Churn](https://www.kaggle.com/blastchar/telco-customer-churn)

**Submodules Covered:**
- Supervised Learning
- Classification vs. Regression
- Model Evaluation Metrics: Accuracy, Precision, Recall, F1 Score, ROC-AUC
- Key Algorithms: Logistic Regression, Decision Trees, Random Forests, Support Vector Machines

**Steps:**
1. Data Preprocessing and Feature Engineering
2. Exploratory Data Analysis
3. Building Classification Models
4. Evaluating Model Performance
5. Hyperparameter Tuning and Model Optimization
